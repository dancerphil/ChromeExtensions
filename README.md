# ZhihuBlocker
让你永远都看不到他！

对已经在你黑名单中的用户进行屏蔽，让他永远不能出现在你的TimeLine,回答和评论中。

首先在 chrome://extensions/ 打开开发者模式，拖动此项目文件夹加载即可使用。
初次使用时会跳转到黑名单页面载入你的名单。
如果你想重载黑名单，你可以在console中输入localStorage.removeItem('UserList');
